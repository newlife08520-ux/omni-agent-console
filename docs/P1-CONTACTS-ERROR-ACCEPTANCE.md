# P1 聯絡人列表錯誤態 — 最短人工驗收

**範圍**：僅驗 `/api/contacts` 失敗時左側不顯示為空列表。  
**預估**：約 2 分鐘。

---

## 步驟

1. **造出失敗情境**  
   - 開 DevTools → Network。  
   - 對 `/api/contacts` 右鍵 → **Block request URL**（或用 Request blocking 加入 `*contacts*`）。  
   - 重新整理首頁（`/`）或切換 viewMode 觸發 request。

2. **預期結果**  
   - 左側列表區出現：「**載入失敗，請重新整理**」+ **重新整理** 按鈕。  
   - 沒有出現「無聯絡人」或「目前沒有分配給你的案件」等空狀態文案。

3. **重試**  
   - 取消 Block，點 **重新整理**。  
   - 左側應恢復為「載入中...」後顯示聯絡人列表或對應空狀態。

4. **收尾**  
   - 關閉 Request blocking，避免影響其他操作。

---

**通過**：失敗時看到錯誤文案+按鈕、重試後恢復即算驗收通過。
