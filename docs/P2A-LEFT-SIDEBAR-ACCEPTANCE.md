# P2-A 左側區塊整理 — 人工驗收步驟

**範圍**：Chat 工作台左側「聯絡人列表」與「對話搜尋結果」資訊結構。  
**不改**：後端、角色、右側、頂欄、指派/改派、訂單查詢。

---

## 1. 沒搜尋時：只有聯絡人列表

- 進入首頁 `/`，搜尋框為空或字數 < 2。
- **預期**：左側僅顯示聯絡人列表（或載入中 / 錯誤 / 依 viewMode 的空狀態）。
- **預期**：沒有「搜尋結果」或「對話紀錄」區塊。

---

## 2. 有搜尋時：清楚區分「搜尋結果」

- 在搜尋框輸入至少 2 個字（例如「測試」）。
- **預期**：左側出現「搜尋結果」標題，底下有：
  - **聯絡人**：名稱符合的聯絡人（若有）
  - **對話紀錄**：對話內容搜尋結果，或「搜尋中...」或「無符合的對話紀錄」

---

## 3. 空狀態區分

| 情境 | 預期 |
|------|------|
| 沒搜尋、列表為空 | 依 viewMode 顯示「目前沒有分配給你的案件」等，無搜尋區塊。 |
| 有搜尋、搜尋中 | 「搜尋結果」→「對話紀錄」底下顯示「搜尋中...」。 |
| 有搜尋、對話無結果 | 「搜尋結果」→「對話紀錄」底下顯示「無符合的對話紀錄」。 |
| 有搜尋、聯絡人無名稱符合 | 仍顯示「搜尋結果」；「聯絡人」區塊不顯示或可為空，「對話紀錄」依結果/搜尋中/無結果顯示。 |

---

## 4. 清除搜尋後回到聯絡人列表

- 有搜尋時點搜尋框右側 X，或點選一筆聯絡人/對話結果進入對話。
- **預期**：左側回到「僅聯絡人列表」，搜尋結果區塊消失。

---

## 5. 回歸（可抽測）

- 切換 viewMode（全部 / 我的案件 / 待我回覆等）、點選聯絡人、開啟對話，行為與改動前一致。
- 聯絡人列表載入中 / 載入失敗狀態與 P1 一致。

---

**通過**：沒搜尋時左側只有聯絡人列表；有搜尋時有「搜尋結果」且聯絡人/對話紀錄標題與空狀態清楚，即算通過。
