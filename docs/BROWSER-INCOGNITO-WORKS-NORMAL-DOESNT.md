# 無痕模式能開、一般模式不好打開 — 說明與處理

若中控台在**無痕／私密視窗**可以正常開啟，但在**一般視窗**卻打不開、白屏或卡住，多半是瀏覽器端的快取或擴充功能造成的。

---

## 建議步驟（依序試）

1. **硬重新整理（清除該頁快取）**  
   - Windows：`Ctrl + Shift + R` 或 `Ctrl + F5`  
   - Mac：`Cmd + Shift + R`  
   - 可多按幾次，確保載入的是最新程式。

2. **清除該網站的網站資料**  
   - Chrome：網址列左側鎖頭／資訊 → 「網站設定」→「清除資料」或刪除 Cookie／快取。  
   - 或 設定 → 隱私權和安全性 → 清除瀏覽資料 → 選擇「快取的圖片和檔案」、時間範圍「不限」，必要時也勾選 Cookie，只針對該中控台網址或「所有時間」清除。

3. **暫時停用擴充功能**  
   - 部分擋廣告、隱私、腳本類擴充會改寫或阻擋前端資源，導致白屏或異常。  
   - 先以「停用所有擴充功能」或開無痕（預設不載入擴充）測試；若停用後一般視窗就正常，再逐個開啟找出有問題的擴充。

4. **改用無痕當作日常使用（權宜之計）**  
   - 若確認是擴充或舊快取造成，短期可先用無痕視窗操作中控台，同時保留上述清除快取／停用擴充的習慣，避免一般視窗再次累積問題。

---

## 為什麼無痕比較容易正常？

- 無痕視窗**不會帶入**一般視窗的舊 Cookie、Local Storage、快取。  
- 無痕**預設不載入**大部分擴充功能。  
- 所以若問題來自「舊版 JS/快取」或「擴充干擾」，無痕就會表現正常。

---

## 若仍無法解決

- 確認網址與環境無誤（例如本機 `localhost` 或 Railway 網址）。  
- 用開發者工具 (F12) → Console 看是否有紅色錯誤，把錯誤訊息提供給維護人員比對。
